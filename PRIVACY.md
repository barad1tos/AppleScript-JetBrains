# Privacy Policy

Last updated: June 8, 2026

AppleScript Toolkit does not sell, share, or transmit personal data to external services.

The plugin runs locally inside JetBrains IDEs. AppleScript execution, application dictionary discovery, and dictionary loading may interact with local macOS applications and files only as part of IDE actions started by the user or by the IDE's local plugin workflow.

To provide dictionary support, the plugin may store local IDE cache data on the user's machine, including discovered or loaded dictionary metadata, generated SDEF/XML dictionary files, application names, and local application or dictionary file paths. This local data is not sent to the developer or to an external service by the plugin.

The plugin does not operate external analytics, telemetry, advertising, or tracking services.

Issue reports, support requests, Marketplace reviews, and other user-submitted messages are handled through GitHub and JetBrains Marketplace. Those services are governed by their own privacy policies.
